# Homepage design system

Direction and feel: showroom-industrial meets finance desk; neutral, confident, human.
Depth strategy: borders + subtle shadows (no dramatic elevation).
Spacing base unit: 4px (components use 8/16/24/32 multiples).

### Section Label
- Text: text-xs, font-semibold, uppercase, tracking 0.28em
- Color: text-primary

### Pill Chip
- Padding: 8px 16px (px-4 py-2)
- Radius: full
- Border: 1px solid border-foreground/10
- Background: bg-foreground/5
- Text: text-xs, font-semibold

### Section Card
- Radius: 24px (rounded-3xl)
- Border: 1px solid border-foreground/10
- Background: bg-background/80 with backdrop blur
- Padding: 24px (p-6)
- Shadow: shadow-sm

### Feature Card
- Radius: 16px (rounded-2xl)
- Border: 1px solid border-foreground/10
- Background: bg-background/70
- Padding: 24px (p-6)
- Shadow: shadow-sm

### Inline Status Dot
- Size: 8px (h-2 w-2)
- Color: bg-primary

### CTA Button
- Width: max-w-72, w-full
- Size: lg (h-11 px-8)
- Font: font-bold, text-base
- Background: bg-primary or bg-black per section

### Section Accent Backgrounds
- Blob: rounded-full, bg-primary/10-25, blur-3xl
- Highlight line: linear-gradient 90deg from rgba(249,115,22,0.12) to transparent, height 32-40px
